Otázka 1:
    Jsem tvá nestálá paměť. Uchovávám celý svět, dokud proudí energie. Jakmile nastane tma,
    zapomenu vše, co jsem věděl. Zadej mou třípísmennou zkratku.
ODPOVĚĎ: RAM

Otázka 2:
    V mém světě existují pouze dva stavy: 1 a 0. Bytí a Nebytí. Všechno vědění je postaveno z
    jejich nekonečných kombinací. Jak se nazývá tento základní jazyk?
ODPOVĚĎ: Binární soustava (možnost B)

Otázka 3:
    Jsem kostra bez kůže. Definuji strukturu a význam, ale ne vzhled. Beze mě je web jen
    chaotický text bez řádu. Který jazyk jsem?
ODPOVĚĎ: HTML (možnost A)

Otázka 4:
    Jeden Byte je nádoba. Aby se naplnila, potřebuje menší střípky dat. Kolik Bitů musíš spojit,
    abys stvořil jeden celistvý Byte?
ODPOVĚĎ: 8 (možnost B)

Otázka 5:
    Jsem všude kolem tebe v tomto prázdnu. Čím více mě je, tím méně vidíš. Co jsem?
ODPOVĚĎ: TMA
